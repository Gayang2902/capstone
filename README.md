# Simple & Beautiful Password Manager

Electron 기반의 비밀번호 관리 프로그램입니다.

사용자가 계정 및 비밀번호를 안전하게 저장하고 관리할 수 있도록 제작하였으며,  
React 기반 UI와 C++ 백엔드 프로세스를 IPC 통신으로 연동하였습니다.

---

## Features

- 계정 및 비밀번호 저장 기능
- 사용자 정보 관리 기능
- Electron ↔ C++ IPC 통신 연동
- 직관적인 UI 제공
